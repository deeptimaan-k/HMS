
## To Run Locally

Clone the project

```bash
  git clone https://github.com/deeptimaan-k/HMS.git
```

Go to the project directory

```bash
  cd Hospital-Management-System
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
## Tech Stack

**Client:** 

- **React**
- **Redux Thunk**
- **Axios**
- **Ant-Designs**

**Server:**

- **Node Js**
- **Mongo DB**
- **Express Js**
- **Render**
- **JWT**
